Javascript Intro

Javascript is a programming language (for computational or mathematical task or communicate with a system)

Javascript is known as scripting language (language that runs on browser).

History

  • Earlier the Javascript was known as ecmascript.
  • This language was introduced in the year 1995 by Brendan Eich.
  • Initially it was made for netscape browser.
  • It was made in only 12-16 days and includes a lot of bugs initially.
  • In 1996 the internet explorer makes its own ecmascript.
  • Ecma international- An organization which sets the protocols for the online communication. (orders to merge both the ecmascript and make the final output which works on every browser)
  • Javascript is a backward language (Its previous bugs were not removed instead new function or features add to enhance the language).
  • On the other hand the forward language removes the previous bugs and improves itself in the next version like java.
  • Google introduced the runtime environment called node.js (It runs the javascript in the system). Node.js = v8 engine (from chrome) + extra features
  • There are other runtime environment as well like deno.js, bun.js.

Step for node.js

  • Step 1: Open chrome and search downlaod node.js
  • Step 2: Download the LTS (long term support) version from prebuilt binaries.
  • Step 3: Go to cmd and type node -v or node --version
