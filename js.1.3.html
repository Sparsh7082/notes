Output

  • Console: means the whole visible screen.
    console.log()
    It prints the output and comes to the new line.
    Example
    console.log("sparsh");
    Output: sparsh
    This is a statement and end with ;
  • process.stdout.write("")
    It prints the output and stays on the same line.
    Example
    process.stdout.write("sparsh ")
    process.stdout.write("gupta")
    Output: sparsh gupta
