JAVASCRIPT

Input or Variables

  • Its like a container or bucket which caries the values
  • Variables are made in the RAM.
  • Example: age = 25
    age is a variable or container
    = is assignment operator (it assign the value in age)
    25 is the value
  • In level 1 javascript we need to put let before the variable
    Example
    let age = 25
  • When there is a number we will write it directly.
    Example
    let age = 25
  • When there is a sequence of character or string write it in a " ".
    Example
    let name = "sparsh"
  • When there is a single character we write it in a ' '.
    Example
    let grade = 'B'
  • Print the variables
    console.log(name);
    Output: "sparsh"

    console.log(age);
    Output: 25

    console.log(grade);
    Output: 'B'
  • Text + variable
    console.log("my name is ",name);
    Output: my name is sparsh
  • Template string
    console.log(`my name is ${name} and my age is ${age}`);
    Output: my name is sparsh and my age is 25
  • process.stdout.write()
    It only prints string not number.
    Example:
    process.stdout.write(name)
    Output: sparsh
    process.stdout.write(age)
    Output: error
