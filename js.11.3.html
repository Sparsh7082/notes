Pre and post increment

+,- => operator

a,b => operand

i++ (here the ++ is unary operator as it works on single operand)

a+b (here the + is binary operator as it works between two operand)

pre increment

example: ++i, --i

first update then use

let x=10

console.log(++x); //Ans=11

console.log(x); //Ans=11

post increment

example: i++, i--

first use then update

let x=10

console.log(x++); //Ans=10

console.log(x); //Ans=11

example

example 1

let a=10

console.log(++a); //Ans:11

console.log(a++); //Ans:11

console.log(a); //Ans:12

example 2

let a=10

console.log(++a + a++);

Ans: 11 + 11 = 22
