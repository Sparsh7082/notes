Arrays []

Array is a linear data structure. As it keeps the data in the linear format.

Example arr = [1,2,3,4,5]

In Javascript arrays are hetrogenous because it can contain anything like number, string, boolean, function, array, etc. in one array.

Array is the index based. The first element is at 0 index.

Array is object type.

Arrays are Mutable which means the values can be changed

Method 1

let arr=[1,2,3,"sparsh",true,function{}]

Method 2

let arr = new Array(1,2,3,"sparsh",true,function{})

Print an element with the help of index. arr = [1,2,3,4,5]

console.log(arr[0])

output: 1

-1 will not work here

OR

console.log(arr.at(0)) (latest version)

output: 1

console.log(arr.at(-1))

output: 5

Changing the value of elements in an array. arr = [1,2,3,4,5]

arr[0] = 10

console.log(arr)

output: [10,2,3,4,5]

Number of elements in an array. arr = [1,2,3,4,5]

console.log(arr.length)

output: 5

Last value of array. arr = [1,2,3,4,5,"sparsh",true]

console.log(arr[arr.length-1])

output: true

Insert a new value in array. arr = [1,2,3,4,5,"sparsh",true]

arr.push(3)

console.log(arr)

output: [1,2,3,4,5,"sparsh",true,3]

Delete the last value of array. arr = [1,2,3,4,5,"sparsh",true]

arr.pop()

console.log(arr)

output: [1,2,3,4,5,"sparsh"]

Insert a value in the first place. arr = [1,2,3,4,5,"sparsh",true]

arr.unshift("Hello")

console.log(arr)

output: ["Hello",1,2,3,4,5,"sparsh",true]

Delete the first element. arr = [1,2,3,4,5,"sparsh",true]

arr.shift()

console.log(arr)

output: [2,3,4,5,"sparsh",true]

Finding an element in an array. arr = [1,2,3,4,5,"sparsh",true]

console.log(arr.includes(3))

output: true

Finding the index of an element in an array. arr = [1,2,3,4,5,"sparsh",true]

console.log(arr.indexOf(3))

output: 2

Finding the index of an element which is not in an array. arr = [1,2,3,4,5,"sparsh",true]

console.log(arr.indexOf(100))

output: -1

indexOf

It gives the index of the first element it found.

for example: arr=[1,2,3,4,5,"sparsh",3,1,2]

console.log(arr.indexOf(1))

output: 0

lastIndexOf

It gives the index of the last element it found.

for example: arr=[1,2,3,4,5,"sparsh",3,1,2]

console.log(arr.indexOf(1))

output: 7

Converting array to string. arr = [1,2,3,4,5,"sparsh",true]

let str = arr.join()

console.log(typeof str)

output: string

here the join plays the role of seperator as well. for example: join("*") puts the * between the elements.

OR

let str = arr.toString() (earlier version)

Reverse an array arr = [1,2,3,4,5]

console.log(arr.reverse())

Output: [5,4,3,2,1]

Sort an array arr = [300,500,100,400]

console.log(arr.sort())

Output: [100,300,400,500]
