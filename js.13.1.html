Array Methods / Functions

array.length

Gives the number of elements in an array.

                let array = [11,22,33,44,55,66,77,88,99]

                console.log(array.length)

                Output: 9

array.push()

Insert the element into last.

                let array = [11,22,33,44,55,66,77,88,99]

                array.push(100);
                console.log(array);

                Output: [11, 22, 33, 44, 55,66, 77, 88, 99, 100]

array.pop()

Remove the last element.

                let array = [11,22,33,44,55,66,77,88,99]

                array.pop()
                console.log(array);

                Output: [11, 22, 33, 44, 55,66, 77, 88]

array.unshift()

Insert the element into first place.

                let array = [11,22,33,44,55,66,77,88,99]

                array.unshift(100)
                console.log(array);

                Output: [100, 11, 22, 33, 44, 55,66, 77, 88]

array.shift()

Remove the first element.

                let array = [11,22,33,44,55,66,77,88,99]

                array.shift()
                console.log(array);

                Output: [22, 33, 44, 55,66, 77, 88]

array.indexOf()

Gives the index of the element.

                let array = [11,22,33,44,55,66,77,88,99]

                console.log(array.indexOf(88));

                Output: 7

If the element is not present in the array then the index will be -1.

                let array = [11,22,33,44,55,66,77,88,99]

                console.log(array.indexOf(100));

                Output: -1

array.at()

Gives the element on the basis of index.

                let array = [11,22,33,44,55,66,77,88,99]

                console.log(array.at(4));

                Output: 55

array.slice(start,end)

It takes the index as parameter.

The first parameter is inclusive and should be smaller than the second parameter.

The second parameter is exclusive and should be greater than the first paramter.

                let array = [11,22,33,44,55,66,77,88,99]

                console.log(array.slice(2,5));

                Output: [ 33, 44, 55 ]

If we take only 1 parameter it will start from it and gives the output till end.

                let array = [11,22,33,44,55,66,77,88,99]

                console.log(array.slice(5));

                Output: [ 66, 77, 88, 99 ]

array.splice(position,remove,add elements)

The first parameter is position where to add or remove.

The second parameter is how many elements should be removed.

The third parameter is elements to add.

                let array = [11,22,33,44,55,66,77,88,99]

                array.splice(3,1,"sparsh")
                
                console.log(array);

                Output: [ 11, 22, 33, 'sparsh', 55, 66, 77, 88, 99 ]
