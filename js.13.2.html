Arrays Traversal

We can traverse an array with the help of index.

work something on each element of array

Callback function- passing one function inside another function as parameter.

For loop

                let names = ["vikas","rohan","puneet","reena","seema","rani","kalu"]

                for(let idx=0;idx<names.length;idx++){
                    console.log(names[idx])
                }

Output:

                vikas
                rohan
                puneet
                reena
                seema
                rani
                kalu

While loop

                let names = ["vikas","rohan","puneet","reena","seema","rani","kalu"]

                let idx=0;
                while(idx<names.length){
                    console.log(names[idx]);
                    idx++;
                }

Output:

                vikas
                rohan
                puneet
                reena
                seema
                rani
                kalu

Do while loop

                let names = ["vikas","rohan","puneet","reena","seema","rani","kalu"]

                let idx = 0;
                do {
                    console.log(names[idx]);
                    idx++;
                } while (idx<names.length);

Output:

                vikas
                rohan
                puneet
                reena
                seema
                rani
                kalu

For of loop

                let names = ["vikas","rohan","puneet","reena","seema","rani","kalu"]

                for(let student of names){
                    console.log(student); 
                }

Output:

                vikas
                rohan
                puneet
                reena
                seema
                rani
                kalu

For each loop

                let names = ["vikas","rohan","puneet","reena","seema","rani","kalu"]

                names.forEach(function fun(student){
                    console.log(student);  
                })

Output:

                vikas
                rohan
                puneet
                reena
                seema
                rani
                kalu
