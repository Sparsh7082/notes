Switch

  • Switch is a case sensitive.
  • Default case is not necessary but applying it is a good practice.
  • If default is not there there will be no output if the case is not match.
  • Must apply break after each case.

Syntax

            switch(statement-expression){
            case "":
                work
            }

Example 1

Without break

                let age = 18;

                switch(age){
                    case 15:
                        console.log("i am 15");
                    case 16:
                        console.log("i am 16");
                    case 17:
                        console.log("i am 17");
                    case 18:
                        console.log("i am 18");
                    case 19:
                        console.log("i am 19");
                    case "sparsh":
                        console.log("i am sparsh");
                    case 'A':
                        console.log("i am char A");
                    case "dice":
                        console.log("i am dice");
                    default:
                        console.log("i am default");


                }

Output

                i am 18
                i am 19
                i am sparsh
                i am char A
                i am dice
                i am default

With break

                let age = 18;

                switch(age){
                    case 15:
                        console.log("i am 15");
                        break;
                    case 16:
                        console.log("i am 16");
                        break;
                    case 17:
                        console.log("i am 17");
                        break;
                    case 18:
                        console.log("i am 18");
                        break;
                    case 19:
                        console.log("i am 19");
                        break;
                    case "sparsh":
                        console.log("i am sparsh");
                        break;
                    case 'A':
                        console.log("i am char A");
                        break;
                    case "dice":
                        console.log("i am dice");
                        break;
                    default:
                        console.log("i am default");


                }
                console.log("I am out");

Output

                i am 18
                I am out

Default

Example 2

Switch is a case sensitive

                let day = "Tuesday";

                switch(day){
                    case "monday":
                        console.log("its monday");
                        break;
                    case "tuesday":
                        console.log("its tuesday");
                        break;
                    case "wednesday":
                        console.log("its wednesday");
                        break;
                    case "thursday":
                        console.log("its thursday");
                        break;
                    case "friday":
                        console.log("its friday");
                        break;
                    case "saturday":
                        console.log("its saturday");
                        break;
                    case "sunday":
                        console.log("its sunday");
                        break;
                    default:
                        console.log("its dry day");       
                }

Output

                its dry day
