Loops

  • Do some work again and again
  • Loops ask for 3 things / statement.
    1. Initiate (runs only one time)
    2. Condition / statement (checks everytime if the initiate value change) (if the condition statement becomes false then it will throw out of the loop) (the condition checks 1 more than the loop runs)
    3. Update
  • If we know that till how many times the loop will run then use the for loop and if we don't know that till how many times the loop will run then use while loop
