For loop

  • Entry control loop- checks the control or condition when we enter into the loop
  • If we know that till how many time we need to run the loop than use for loop

Syntax

                for(initiate; condition / statement; update){
                    work
                }

Example

                for(let i=1;i<=10;i=i+1){
                console.log("sparsh");
                }
