While loop

  • Entry control loop- checks the control or condition when we enter into the loop
  • if we don't know that till how many times the loop will run then use while loop

Syntax

            initiate

            while(condition / statement){
                work
                update
            }

Example

                let i = 1;

                while(i<=10){
                    console.log("sparsh",i);
                    i=i+1;
                }

Output

                sparsh 1
                sparsh 2
                sparsh 3
                sparsh 4
                sparsh 5
                sparsh 6
                sparsh 7
                sparsh 8
                sparsh 9
                sparsh 10
