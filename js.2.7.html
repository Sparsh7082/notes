Do While loop

  • Exit control loop- It checks the control or condition on exit which means the loop will run atleast 1 time.

Syntax

            initiate

            do{
                work
                update
            }
            while(condition / statement);

Example

                let i = 1;

                do{
                    console.log("sparsh",i);
                    i=i+1;
                }
                while(i<=10);

Output

                sparsh 1
                sparsh 2
                sparsh 3
                sparsh 4
                sparsh 5
                sparsh 6
                sparsh 7
                sparsh 8
                sparsh 9
                sparsh 10
