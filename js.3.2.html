Break and continue

Break

  • Reserved keyword of javascript
  • Throw out of nearest loop
  • apne paas wale loop se bahar fenkega

Continue

  • Reserved keyword of javascript
  • Skip the iteration of loop
  • wo wali iteration skip karega

Example

for(let i=1;i<=10;i=i+1){
    if(i==8){
        break;
    }
    console.log(i);
}

Output

    1
    2
    3
    4
    5
    6
    7

breaks the loop at i=8

Example

for(let i=1;i<=10;i=i+1){
    if(i==5){
        continue;
    }
    console.log(i);
}

Output

    1
    2
    3
    4
    6
    7
    8
    9
    10

Skip the iteration when i=5
