To print an output

console.log("")

  • It prints an output and enter into a new line
    Example
    console.log("sparsh")
    console.log("gupta")

    Output
    sparsh
    gupta
  • we can put anything in console.log() like number, function, etc.

process.stdout.write("")

  • It prints an output and be there in the same line
    Example
    process.stdout.write("sparsh")
    process.stdout.write("gupta")

    Output
    sparshgupta
  • we can put only string value or sequence of character in the process.stdout.write()
