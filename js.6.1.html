Alpha-numeric

  • System only understand binary code which is 0 & 1.
  • With the help of ASCII code system understand all other things.
  • ASCII code- American Standard Code for Information Interchange.
  • Example-
    • 65 for A
    • 97 for a
    • 32 for space

String to number

  • charCodeAt() return the corresponding ascii code.
    Example-
    let name = "sparsh";
    console.log(name.charCodeAt(2))
    Output 97

Number to string

  • String.fromCharCode() converts the ascii code to string.
    Example
    let num=65;
    console.log(String.fromCharCode(num));
    Output A

Q2-structure

                
let nsp = 0;
for(let row=1;row<=5;row++){
    let ans = "";
    //space
    for(let space=1;space<=nsp;space++){
        ans = ans + " ";
    }
    //star
    for(let star=1;star<=5-row+1;star++){
        ans = ans + "*";
    }
    //new line
    console.log(ans);    
    //update
    nsp=nsp+1;
}
