Prime number and approach

  • Prime number are those number who has only 2 factors the number and 1.

Approach 1 (1 -> num)

Here machine runs 100 times

                let num = 100;
                let nof = 0;
                for(let div=1;div<=num;div++){
                    if(num%div==0){
                        nof++;
                    }
                }
                if(nof>2){
                    console.log("not prime");
                }
                else{
                    console.log("prime");
                }
